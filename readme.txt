To install the ELEGOO Mars 5 profile for Voxeldance Tango, just download the files and copy them 
to the "C:\Program Files\Voxeldance Tango\Voxeldance Tango 4.0\labs\ELEGOO", or the "labs/ELEGOO" 
folder wherever you have it installed. In the event that theversion you have isn't 4.0, then 
replace it with whatever the new version is. You got this.
